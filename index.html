Paper Year Input Output Pro Con
MaterialMVP 2025 3D mesh + image prompt PBR texture maps: albedo, metallic, roughness Good at separating color from roughness/metallic instead of baking lighting into one texture Needs a clean mesh/UV, and it does not decide where object parts should be
Hunyuan3D 2.1 2025 Single image Textured 3D asset with mesh and PBR textures Handy when there is only one photo and we need a complete textured asset It regenerates the object, so small parts and topology can drift
Meshy N/A Text, image, or uploaded mesh Export-ready textured 3D model or retextured mesh Fast and convenient for getting a previewable textured model Closed service, so exact behavior and part preservation are hard to inspect
TRELLIS 2024 Text or image prompt 3D asset decoded as mesh, 3D Gaussian, or radiance field Can produce several 3D representations from the same prompt or image More about generating a new asset than keeping a given multipart mesh unchanged
Paint3D 2024 Untextured mesh + text or image prompt Lighting-less 2K UV texture map Removes much of the lighting baked into view-based textures Mainly outputs RGB texture and still depends on the given UV/mesh quality
TEXGen 2024 Mesh + text, single image, or sparse views High-resolution UV texture map Works in UV space, which helps with completion and seam handling Does not really reason about materials or object parts beyond the texture map
MVPaint 2025 3D mesh + text prompt High-resolution seamless UV texture Synchronized views make the painted texture more consistent around the object Still a multi-stage RGB texture pipeline, without geometry or PBR decomposition
TexPainter 2024 3D mesh + text prompt Multi-view-consistent mesh texture Color fusion helps when different rendered views disagree Per-object optimization costs time and does not add material maps
SyncMVD 2024 3D mesh + text prompt Consistent UV texture for the input object Shares the texture during denoising, so different views fight less Fixed camera views can still miss hidden or thin parts
TexGen 2024 3D mesh + text prompt RGB UV texture map Sampling and resampling across views helps reduce seams Can still paint details that do not quite fit the geometry
Text2Tex 2023 3D mesh + text prompt Textured mesh / UV texture Simple progressive painting strategy; easy to understand and compare against Sequential painting can accumulate seams and view-to-view mismatch
TEXTure 2023 3D shape + text, image, or scribble guidance Generated, edited, or transferred texture Covers texture generation, editing, and transfer in one framework Iterative view painting may bake lighting and inconsistent details
MVPainter 2025 Reference image + known 3D geometry PBR textured mesh Uses geometry control to keep generated views closer to the reference Needs a good reference and has more moving parts in the pipeline
Material Anything 2025 Textured or texture-less 3D object UV-ready PBR material maps Useful for turning an existing object into cleaner PBR material maps Material-focused; it does not solve part segmentation or geometry layout
MeshGen 2025 Single image PBR textured 3D mesh Builds both shape and PBR texture from one image Can change the target topology, so it is not ideal for preserving multipart meshes
